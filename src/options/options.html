x-true-block-mute 設定

取り込んだデータの確認・管理と、プライバシーについての説明です。

プライバシー

この拡張がどのデータを、どこに保存するかの説明です。

  • データはすべてこの端末内（chrome.storage.local）に保存され、外部サーバーへは送信されません。
  • 取り込むのはブロック・ミュート対象の user_id・handle・一覧種別（blocked / muted）のみです。表示名・投稿本文・cursor・Cookie・token は保存しません。
  • 使用する権限は storage と x.com / twitter.com への接続のみです。
  • 取り込んだデータは下の「同期データを消す」でいつでも削除できます。

フィルタ対象（同期で取り込んだ一覧）

同期で取り込んだブロック・ミュート対象の一覧です。すべてこの端末内のデータです。

ブロック 0件
ミュート 0件
ローカル確認用データ 0件

    同期で取り込んだ対象はまだありません。popup から同期を有効にして、ブロック / ミュート一覧を開いてください。

    管理

    取り込んだデータや、ローカル確認用のテストデータを削除できます。

    うまく同期できないとき

    件数が増えないときは、次の順に確認してください。

    1. popup で「同期を有効にする」が ON か確認
    2. ブロック / ミュート一覧を一番下までスクロール
    3. ページを再読み込み
    4. この設定画面を再読み込み

    拡張として読み込めていない場合は chrome://extensions で再読み込みしてください。
